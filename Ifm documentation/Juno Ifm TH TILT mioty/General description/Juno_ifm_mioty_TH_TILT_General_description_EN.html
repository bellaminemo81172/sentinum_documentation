2. Overview

The Juno IP67 TH is a versatile temperature and humidity sensor featuring mioty® wireless technology for reliable and energy-efficient data transmission over long distances. Thanks to its robust design and IP67 protection, it is ideal for use in demanding environments. Additionally, the sensor features tilt and damper opening detection with degree-precise adjustment.

3. Features

  • mioty® wireless technology
  • IP67, -25 °C to 85 °C, IK07
  • High-precision temperature and relative humidity sensor
  • Tilt and flap opening detection
  • Advanced tilt detection with 1° resolution and thresholds
  • Magnetic switch
  • Can be activated with a magnet (no smartphone required) or via NFC
  • Configurable via NFC
  • Adjustable thresholds
  • Replaceable AA cells (LiSOCl₂, see Saft LS14500)
  • Audible signal

4. Example Applications

  • Temperature and relative humidity monitoring in industry
  • Cold chain monitoring
  • Temperature monitoring in building management or smart city applications
  • Control cabinet monitoring
  • Manhole cover monitoring
  • Damper opening detection
