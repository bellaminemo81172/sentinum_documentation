2. Einbindung von Blueprint Sensoren in das ifm mioty Gateway

Schritt 1

Nehmen Sie das Gateway in Betrieb. Halten Sie sich dabei an die von ifm bereitgestellten Dokumente. Im Folgenden sollten Sie folgenden Bildschirmausschnitt sehen.

Schritt 2

Navigieren Sie zum Reiter „Data Interpretation“ und hier zum Reiter „Blueprint“.

Schritt 3

Laden Sie sich den Blueprint für Ihren ausgewählten Sensor herunter. Navigieren Sie dazu auf die Seite https://docs.sentinum.de und suchen Sie nach Ihrem Sensor und dem entsprechenden Blueprint. In diesem Fall legen wir einen Juno Temperatur- und rel. Luftfeuchtigkeitssensor an.

Schritt 4

Klicken Sie auf den Eintrag und dann auf die Schaltfläche JSON-Datei. Alternativ kann weiter unten auch eine TXT-Datei heruntergeladen werden.

Schritt 5

Jetzt sollte sich der Blueprint öffnen. Speichern Sie diese Datei mit einem Rechtsklick und „Speichern unter…“ in einem Ordner ab.

Schritt 6

Springen Sie jetzt wieder zurück in die Oberfläche des Gateways. Klicken Sie im Reiter „Data Interpretation“ und im Reiter „Blueprint“ auf die Schaltfläche „+ Add New…“. Laden Sie jetzt die gespeicherte Blueprint-Datei als .json hoch.

Schritt 7

Jetzt sollten Sie einen neu angelegten Blueprint sehen, in diesem Fall Juno. Dem Blueprint wird automatisch eine EUI zugewiesen. Kopieren oder speichern Sie diese EUI, da sie später nochmals benötigt wird.

Schritt 8

Binden wir jetzt den Sensor ein. Navigieren Sie dazu in den Reiter mioty auf der Gateway-Oberfläche und klicken Sie auf „+ Attach New“.

Schritt 9

Jetzt erscheint folgendes Fenster. Klicken Sie oben in der Checkbox auf Blueprint, sodass Sie unten das Eingabefeld Blueprint EUI sehen. Geben Sie hier die nötigen Informationen ein und klicken Sie dann auf „OK“. Diese finden Sie in der Verpackung als QR-Code oder Sie fordern diese bequem als CSV-Datei bei info@sentinum.de an. Wichtig: Bei diesem Schritt benötigen Sie die Blueprint EUI wieder.

Schritt 10

Jetzt haben Sie den Sensor erfolgreich hinzugefügt.

Schritt 11

Sie sollten jetzt den Sensor mit allen nötigen Informationen sehen.

Schritt 11

Aktivieren Sie den Sensor jetzt mit Magnet oder NFC. Warten Sie danach eine kurze Zeit und betätigen Sie die Schaltfläche „Refresh“. Dann sollten Sie folgende Werte sehen können.

Schritt 12

Mit einem Klick auf den Graph rechts in der Aktionsspalte können Sie die Sensorwerte einsehen.
