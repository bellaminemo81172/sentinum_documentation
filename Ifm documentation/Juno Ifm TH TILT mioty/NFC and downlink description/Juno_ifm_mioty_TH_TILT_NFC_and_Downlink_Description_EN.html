2. Module Overview

Module Module Key Group Group ID Description
Sentiface 0x1XXX Timings 0x00 Defines parameters for measurements and transmission behavior
Sentiface 0x1XXX Alarms and thresholds 0x01 Defines alarms and limit values, such as upper and lower temperature limits
Sentiface 0x1XXX Settings for opening detection 0x02 Defines measurement thresholds for triggering the alarm for temperature and relative humidity
Sentiface 0x1XXX Settings for tilt detection 0x03 Defines the behavior of the accelerometer

Tables for cross-product modules (Senticom/Sentivisor) can be found in the Generic NFC and Downlink documentation.

For more information on configuring sensor communication, refer to the respective mioty® documentation depending on the version.

3. Sentiface: Timings Group 0x00

Resources Resource ID Description Key (NFC/BLE) Min Max Factory Setting Unit Module Key
MEASUREMENT PERIOD 0x00 Specifies the interval at which measurements are taken. 5 minutes means that the sensor takes a measurement every 5 minutes. period 1 360 30 min 1111
REGULAR TRANSMISSION INTERVAL 0x01 Number of measurements performed prior to transmission. every 1 64 6 1111

4. Sentiface: Alarm Settings Group 0x01

Resources Resource ID Description Key (NFC/BLE) Min Max Factory Setting Unit Module Key
ALARM ACTIVE 0x00 Indicates which alarms, whether for temperature or relative humidity, are active:
0: Both alarms deactivated
1: Temperature alarm active
2: Relative humidity alarm active
3: Both alarms active 		aact 0 3 1 1111
TEMPERATURE DELTA 0x01 Specifies the absolute change in temperature compared to the last transmission that must occur to trigger an ALARM transmission. This transmission is performed independently of the normal transmission interval. tdt 1 50 5 °C 1111
UPPER TEMPERATURE LIMIT 0x02 Specifies the upper absolute limit at which an alarm is triggered. tht 0 85 40 °C 1111
TEMPERATURE BELOW LIMIT 0x03 Specifies the lower absolute limit at which an alarm is triggered. tlt -25 75 0 °C 1111
RELATIVE HUMIDITY DELTA 0x04 Specifies the absolute change in relative humidity compared to the last transmission that must occur in order to trigger an ALARM transmission. This transmission is performed independently of the normal transmission interval. hdt 1 80 20 %RH 1111
RELATIVE HUMIDITY UPPER LIMIT 0x05 Specifies the upper absolute limit at which an alarm is triggered. hht 5 100 80 %RH 1111
RELATIVE HUMIDITY BELOW THRESHOLD 0x06 Specifies the lower absolute limit at which an alarm is triggered. hlt 0 95 20 %RH 1111
TEMPERATURE HYSTERESIS 0x07 Determines the absolute hysteresis in °C during state changes. If the temperature falls below or exceeds a threshold value minus the hysteresis, the system switches from a higher to a lower alarm state. thy 0 10 2 °C 1111
HYSTERESIS REL. HUMIDITY 0x08 Determines the absolute hysteresis in %RH during state changes. If the relative humidity falls below or exceeds a threshold value minus the hysteresis, the system switches from a higher to a lower alarm state. hhy 0 30 5 %RH 1111

5. Sentiface: Opening Detection Group 0x02

Property Property ID Description Key (NFC/BLE) Min Max Default Unit Module Key
OPENING COOLDOWN 0x00 Specifies a time period during which, after an opening has been triggered, another opening cannot be triggered (de-bounces the openings). acool 0 600 0 sec 1111
OPENING TIME ALARM 0x01 Specifies how long the lid must remain open before an alarm is triggered (0 = immediate alarm). oaaf 0 2880 0 sec 1111
OPENING MODE 0x02 Describes which direction the sensor faces when closed and which angles are set for opening detection. Settings 0–7 use the accelerometer; 8 and 9 use the Hall sensor:
0: off
1: Ultra Low Power. Fixed threshold 50°. Closed when facing down (Face Down).
2: Ultra Low Power. Fixed threshold 50°. Closed when facing up (Face Up).
3: Ultra Low Power. Fixed threshold 50°. Closed when facing sideways.
4: Advanced tilt sensing. Adjustable threshold. Closed when facing down (Face Down).
5: Advanced tilt sensing. Adjustable threshold. Closed when facing up (Face Up).
6: Advanced tilt sensing. Adjustable threshold. Closed when facing sideways.
7: Advanced tilt sensing. Adjustable threshold. Automatic calibration of closed orientation.
8: Container is closed when the magnet is in contact.
9: Container is open when magnet is present. 		opmo 0 9 1 1111

6. Sentiface: Advanced Tilt Settings Group 0x03

Property Property ID Description Key (NFC/BLE) Min Max Default Unit Module Key
SAMPLING PERIOD 0x00 Specifies how often the accelerometer determines the angle. The higher the frequency, the higher the power consumption. tspe 2 600 2 s 1111
TRIGGER LEVEL 0x01 Specifies the angle at which an opening is detected or an alarm is triggered. ttle 5 180 20 ° 1111
TRIGGER HYSTERESIS 0x02 Hysteresis value for the opening angle. tthy 1 90 4 ° 1111
