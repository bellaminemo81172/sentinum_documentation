1. Payload description

This chapter describes the structure of the telemetry data. The number of bytes in the payload depends on the configuration of the sensor. The structure of the data depends on which sensors are present in your product.

In principle, each version has a header. This provides information about the version and the status of the sensor and also contains a master measured value. This value can be regarded as the current measured value.

The header is followed by an additional payload section containing further information, such as parameters of the measuring principle or position and opening detection settings. Not every sensor has an additional payload part.
