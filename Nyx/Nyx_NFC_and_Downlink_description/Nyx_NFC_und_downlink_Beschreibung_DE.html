2. Gruppe Timings 0x00

Property Property ID Beschreibung Key (NFC/BLE) Min Max Default Unit Module Key
MESSPERIODE 0x00 Gibt die Periode an, in der die Messwerte erfasst werden. 5 Minuten bedeutet, dass immer nach 5 Minuten eine Messung durch den Sensor durchgeführt wird. period 5 360 30 min 1111
REGULÄRES SENDEINTERVALL 0x01 Anzahl der bis zur Übertragung durchgeführten Messungen. every 1 64 8 - 1111

3. Gruppe Alarm Einstellungen 0x01

Property Property ID Beschreibung Key (NFC/BLE) Min Max Default Unit Module Key
DELTA GRENZWERT 0x00 Gibt an, um wie viel sich der CO₂-Wert im Vergleich zur letzten Übertragung absolut ändern muss, damit eine ALARM-Übertragung ausgelöst wird. Diese Übertragung wird unabhängig vom normalen Übertragungsintervall durchgeführt. delta 300 2000 300 ppm 1111
GELBER GRENZWERT 0x01 Gibt die CO₂-Schwelle an, die überschritten werden muss, damit der Sensor in den gelben Zustand wechselt (gelbe LED leuchtet). yellow 500 3000 1000 ppm 1111
ROTER GRENZWERT 0x02 Gibt den CO₂-Schwellenwert an, der überschritten werden muss, damit der Sensor in den roten Zustand wechselt (rote LED leuchtet). red 500 3000 2000 ppm 1111
LEVEL HYSTERESE 0x03 Bestimmt die absolute Hysterese für Zustandsänderungen. Unterschreitet die CO₂-Konzentration einen Schwellenwert minus Hysterese, wechselt das System von einem höheren in einen niedrigeren Alarmzustand. hyst 0 300 100 ppm 1111

4. Gruppe Buzzer0x02

Property Property ID Beschreibung Key (NFC/BLE) Min Max Default Unit Module Key
BUZZER EINSTELLUNG 0x00 Legt das Verhalten des Buzzers fest.
0: Buzzer aus
1: Buzzer bei Änderung des Alarmpegels oberhalb der festgelegten Schwellenwerte (rot, gelb, grün)
2: Immer eingeschaltet im Falle eines Alarms (bleibt der Sensor im gelben oder roten Zustand, meldet der Summer bei jeder Messung den Zustand)
3: Buzzer meldet nur beim Wechsel in den roten Zustand
4: Buzzer meldet permanent, wenn der rote Zustand bestehen bleibt
5: Buzzer meldet permanent, wenn er sich nicht im grünen Bereich befindet und eine Pegeländerung oberhalb der definierten Schwellenwerte auftritt 		buzz 0 5 1 - 1111

5. Gruppe LED 0x04

Property Property ID Beschreibung Key (NFC/BLE) Min Max Default Unit Module Key
LED EINSTELLUNG 0x00 Legt das Verhalten der LEDs fest:
0: Alle LEDs aus
1: Alle LEDs an 		leds 0 1 0 - 1111

6. Gruppe Feuchtigkeitsalarm 0x05

Property Property ID Beschreibung Key (NFC/BLE) Min Max Default Unit Module Key
ALARM FEUCHTIGKEIT 0x00 Schaltet den Alarm für Luftfeuchtigkeit bei Status 1 ein, bei Status 0 aus. ads 0 1 0 - 1111
DELTA GRENZWERT 0x01 Gibt den absoluten Betrag an, um den sich die Luftfeuchtigkeit im Vergleich zur letzten Übertragung ändern muss, damit eine ALARM-Übertragung ausgelöst wird. Diese Übertragung wird unabhängig vom normalen Übertragungsintervall durchgeführt. hdt 5 100 20 %RH 1111
GELBER GRENZWERT 0x02 Gibt die Feuchtigkeitsschwelle an, die überschritten werden muss, damit der Sensor in den gelben Zustand wechselt (gelbe LED leuchtet). hyt 10 100 60 %RH 1111
ROTER GRENZWERT 0x03 Gibt die Feuchtigkeitsschwelle an, die überschritten werden muss, damit der Sensor in den roten Zustand wechselt (rote LED leuchtet). hrt 10 100 80 %RH 1111
LEVEL HYSTERESE 0x04 Bestimmt die absolute Hysterese für Zustandsänderungen. Unterschreitet die Luftfeuchtigkeit einen Schwellenwert minus Hysterese, wechselt das System von einem höheren in einen niedrigeren Alarmzustand. hlh 0 300 5 %RH 1111
